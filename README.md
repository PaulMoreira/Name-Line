    <h1>Name Line</h1>
    <p>Name Line is an engaging and interactive trivia web app that tests users' knowledge of famous personalities across different domains. Built using JavaScript, HTML, and CSS, this app leverages the power of Firebase to manage the database of questions and answers.</p>

<h2>Features</h2>

<ul>
    <li>Dynamic and randomized questions for an engaging experience</li>
    <li>Countdown timer to challenge users</li>
    <li>Displays correct answers when answered incorrectly</li>
    <li>Score tracking system</li>
    <li>Responsive design for seamless experience on multiple devices</li>
</ul>

<h2>Getting Started</h2>

<p>To get started with the Name Line web app, simply clone the repository and follow the instructions below:</p>

<h3>Prerequisites</h3>

<ul>
    <li>A web browser (e.g., Chrome, Firefox, Safari)</li>
    <li>A text editor (e.g., VS Code, Sublime Text, Atom)</li>
</ul>

<h3>Installation</h3>

<ol>
    <li>Clone the repository to your local machine:
        <pre><code>git clone https://github.com/yourusername/Name-Line.git</code></pre>
    </li>
    <li>Open the <code>index.html</code> file in your preferred web browser.</li>
    <li>Enjoy the trivia experience!</li>
</ol>

Name Line

Name Line is an engaging and interactive trivia web app that tests users' knowledge of famous personalities across different domains. Built using JavaScript, HTML, and CSS, this app leverages the power of Firebase to manage the database of questions and answers.

Features
Dynamic and randomized questions for an engaging experience
Countdown timer to challenge users
Displays correct answers when answered incorrectly
Score tracking system
Responsive design for seamless experience on multiple devices
Getting Started
To get started with the Name Line web app, simply clone the repository and follow the instructions below:

Prerequisites
A web browser (e.g., Chrome, Firefox, Safari)
A text editor (e.g., VS Code, Sublime Text, Atom)

Installation
Clone the repository to your local machine:
git clone https://github.com/yourusername/Name-Line.git
Open the index.html file in your preferred web browser.

Enjoy the trivia experience!
